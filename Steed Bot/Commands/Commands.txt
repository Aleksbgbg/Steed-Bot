help
!help
^!help$
Outputs a list of command queries supported by Steed Bot.

steed
!steed [changelog] [version] [update]
^!steed (?<Parameter>changelog|version|update)$
Downloads and outputs a message containing the latest changelog file; Steed release date; or Steed update available on the Steed servers, as specified. At least one parameter must be specified.
